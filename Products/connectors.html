Механические соединители

Предлагаем вашему вниманию механические соединители, представляющие собой высокотехнологичные шарнирные соединения.

Данная методика стыковки бесконечных лент позволяет соединять резинотканевые и ПВХ конвейерные ленты, приводные ремни любой толщины (от 0,8 мм) и прочности (до 3500 Н/мм).

Использование этой технологии стыковки позволяет сократить расход конвейерной ленты на стык (экономится до двух метров ленты, а значит снижаются расходы на её приобретение), сократить время на стыковку (бригада из двух человек изготавливает стык за 20-30 мин.), нет необходимости в подготовке ленты (простая обрезка под прямым углом).

Полученный стык является разъемным (упрощается монтаж, демонтаж ленты) и имеет гибкость на уровне цельной ленты. Одинаковая по всей толщине лента не затруднит прохождение через очистные устройства. Этот метод стыковки просто незаменим при различных авариях на конвейерах.

С описанием и испытаниями надежности и долговечности механических соединителей можно ознакомиться перейдя по этой ссылке

К-28 толщина ленты 10-13 мм

Примечание: поставляется по заказу потребителя в зависимости от ширины соединяемой ленты.

Разъемные механические соединители конвейерных лент УМ

Скобы УМ1000.10, УМ1400.13, УМ1750.15 прочно запрессовываются в конвейерную ленту при помощи специального устройства УМ130.

Для соединения концов конвейерной ленты после запрессовки соединителей используется соединительный стержень УМ или бронированный соединительный стержень УМ Б.

Разъемные механические соединители DR (МАТО Германия)- аналог FLEXCO ALLIGATOR RS.

Пластинчатые соединители предназначены для стыковки конвейерных лент толщиной от 1,5 до 6,4 мм.

Для быстрого и качественного изготовления стыкового соединения конвейерной фирма есть специальное устройство для монтажа соединителей DR, с помощью которого можно состыковать ленту любой ширины.

Выбор соединителя производится по следующим параметрам:

  • толщина ленты,
  • ширина ленты,
  • отсутствие барабана,
  • перевозимый материал.

Соединители DR изготавливаются двух типов, в зависимости от области применения:

  • для агрессивных сред из нержавеющей стали «SS» 1.4404;
  • для обычных условий из стали оцинкованной «G».

Механические соединители конвейерных лент РШМ 120/1600 (аналог "Вулкан"- Круг 120/1600 )

Предназначены для стыковки конвейерных лент толщиной от 8 до 20 мм без специальных устройств.

Ремонтная винтовая система MS

Предназначена для стыковки и ремонта продольных порывов резинотканевых и резинотросовых конвейерных лент.

Дополнительное оборудование

Угольник L = 1000 мм

Угольник L = 1200 мм

Угольник L = 1400 мм

Описание: Угольник предназначен для обрезки ленты под прямым углом. Имеет Т-образный катет для упора в борт ленты, прорезь для точного позиционирования ножа и свинцовые вставки на концах прорези для предотвращения поломки лезвия ножа. Имеет антикоррозийное покрытие.

Специальный нож

Описание: Нож специальный предназначен для отрезки и снятия рабочей обкладки конвейерной ленты вместе с угольником. Имеет сменное сегментированное лезвие, не требующее заточки, обрезиненную ручку, механический фиксатор лезвия.

Канавконарезатель

Описание: Канавконарезатель предназначен для снятия части обкладки конвейерной ленты при ее толщине свыше 2 мм.
